# Torno Los Güichos



## Archivos

- `index.html` — página principal del sitio.
- `img/` — carpeta con el logo y otros recursos de imagen.

## Publicar en GitHub Pages

1. Abre el repositorio en GitHub: `https://github.com/faviandarkrose-code/Torno-los-guichos`
2. Ve a `Settings` > `Pages`.
3. En `Build and deployment`, selecciona:
   - `Branch`: `main`
   - `Folder`: `/ (root)`
4. Guarda.

Después de unos minutos, tu sitio estará disponible en una URL como:

`https://faviandarkrose-code.github.io/Torno-los-guichos/`
